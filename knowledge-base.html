Detection Knowledge

SecureAI Sentinel currently detects suspicious behavior from uploaded CSV logs using basic defensive rules. These rules will later be improved using AI and RAG-based context.

Threat Type Detection Logic Security Meaning
Brute-force Login Attempt Many failed login attempts from the same IP Possible password guessing or credential attack
Suspicious API Access Repeated 401, 403, or 404 responses on API endpoints Possible API probing or unauthorized access attempt
Possible Data Exfiltration Large data download or abnormal bytes sent Possible unauthorized data transfer
Sensitive Data Exposure Logs showing debug exposure or sensitive data indicators Possible leakage of secrets, tokens, or private information
